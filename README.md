# sm_rules
A "better" custom rules plugin!
Create your customizable rule menus with ease.

## How to Install
- Put ``sm_rules.smx`` to ``addons/sourcemod/plugins``
- Put ``smrules.cfg`` to ``addons/sourcemod/configs``
- Put ``smrules.phrases.txt`` to ``addons/sourcemod/translations``


## Usage
- ``sm_update_rules`` admin command: Use to update rules in-game.
- Configure ``smrules.cfg`` "Settings" to change menu name, command and showing menu on connect.
- Configure ``smrules.cfg`` To create rule heads and articles.

## Other Features
- Add "ITEMDRAW_SPACER" as an article to create spacing.
